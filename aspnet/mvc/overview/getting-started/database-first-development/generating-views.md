# <a name="tutorial-generate-views-for-ef-database-first-with-aspnet-mvc-app"></a>チュートリアル: ASP.NET MVC アプリで EF Database First のビューを生成します。

このチュートリアルでは、ASP.NET のスキャフォールディングを使用して、コント ローラーとビューを生成する重点を置いています。
